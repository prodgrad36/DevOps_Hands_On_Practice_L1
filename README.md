# DevOps_Hands_On_Practice_L1
This repo provides hands-on, foundational code to accelerate your DevOps skills with industry-standard tools like Ansible, Terraform, Kubernetes, Jenkins, and GitLab. Inside, you’ll find practical examples and configurations to automate infrastructure, container management, and streamline CI/CD pipelines, designed to enhance your DevOps skills.
# Prodgrad 30-Day DevOps Hands-On Challenge(New joiners can start working on it from the day they learn GIT)
This repository contains a complete 30-day practical DevOps learning challenge.
Each day includes:

Ansible YAML task
Terraform AWS EC2 task
Kubernetes YAML task
Shell scripting task
Jenkins Pipeline task
GitLab CI/CD pipeline task
Full example code (students must retype manually)
EC2 instance types used:

c5.xlarge
r5.2xlarge
r6i.4xlarge
c6i.8xlarge
c6a.12xlarge
Naming convention:
Everything uses "Prodgrad" (e.g., ProdgradEC2, ProdgradVPC, ProdgradDeployment, Prodgrad-Training).

Environment variables used in pipelines:

INSTITUTE_NAME = "Prodgrad"
TEAM = "DevOps"
ENV = "prod"
PROJECT = "Prodgrad-Training"
Students must rewrite all code from scratch to improve speed, understanding, and real-world readiness.
